# HealthTech Reboot 

Projet d'apprentissage Python & SQL orienté HealthTech.

## Objectifs

- Développer des compétences Python pour l'analyse de données santé
- Créer et manipuler une base PostgreSQL
- Comprendre le workflow Git/GitHub pour la gestion de projets

## Contenu 

### Python 
- Scripts pour l'analyse de mesures cardiaques
- Exercices sur input(), boucles, conversions de type et f-strings

### SQL 
- Création de tables
- Insertion et sélection de données
- Requêtes avec WHERE, ORDER BY et LIMIT

### git
- Historique complet des commits
- Projet en ligne sur GitHub
